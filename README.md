# 🚀 Zexa AI

Zexa AI is a modern multi-model AI chat application built using **FastAPI**, **HTML/CSS/JavaScript**, and local LLM integrations like **LLaMA 3**, **Phi-3**, **Mistral**, and **Qwen**.

The application provides:

- 🧠 Multiple LLM model selection
- ⚡ Automatic best-model detection
- 💬 Real-time AI conversations
- 📊 Session statistics
- 🎨 Modern responsive UI
- 🔥 FastAPI backend
- 🖥️ Local AI inference support

# ✨ Features

## 🤖 Multi-Model AI Support

Users can manually choose between:

- LLaMA 3
- Phi-3
- Mistral
- Gemma 2
- Qwen 2.5

## ⚡ Automatic Model Selection

Zexa AI intelligently selects the best LLM based on:

- Coding tasks
- Long-context tasks
- Creative writing
- General conversation
- Logical reasoning
